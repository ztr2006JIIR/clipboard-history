# 历史粘贴板

这是一个本地 Windows 小工具：运行后会监听你复制的内容，并把最近 3 天的历史保存在本文件夹的 `data` 目录里。

## 怎么使用

1. 推荐双击 `Start-ClipboardHistory.vbs`，它会静默启动，不显示 PowerShell 黑窗口。也可以双击 `Start-ClipboardHistory.bat` 启动。
2. 启动后屏幕上会出现一个写着「剪」的小圆圈。
3. 平时不用时，让这个小圆圈留在屏幕上即可，它会继续监听你复制的内容。
4. 按住小圆圈可以拖动位置；拖到屏幕边缘附近松手，它会自动吸附成半圆。
5. 单击小圆圈，会展开「历史粘贴板」悬浮框。
6. 只要工具仍在后台运行，快速单独按两次左 `Ctrl` 或右 `Ctrl`，也会直接执行托盘菜单里的「打开历史」。窗口会移动到你当前所在的 Windows 虚拟桌面；正常的 `Ctrl+C`、`Ctrl+V` 等组合键不会触发。
7. 你正常复制文字、图片或文件，它会自动出现在左侧黑蓝预览区；图片会直接显示缩略图，文字会直接显示内容摘要。
8. 左侧有「全部」「图片」「文字」三个按钮。点「图片」会切换成图片矩阵，方便一次看很多张图；点「文字」只看文字记录。
9. 想恢复某一条内容时，先点左侧那条记录，再点「复制回剪贴板」。右侧仍然会显示更大的预览。
10. 在「图片」矩阵里可以按住 `Ctrl` 点选多张图片，或按住 `Shift` 连续选择一段图片；再点「复制回剪贴板」会把这些图片作为可粘贴的文件组稳定写入系统剪贴板。看到“可切换到目标位置直接粘贴”的提示后，再到文件夹、聊天窗口或文档里按 `Ctrl+V`。
11. 也可以直接拖动复制：从左侧列表、左侧图片矩阵、右侧文字框或右侧大图按住拖出去，放到微信、Word、浏览器等支持拖放的软件里；图片矩阵多选时会一起拖出多张图片。
12. 左侧历史列表和右侧预览区中间有一条竖向分隔条，按住拖动可以调整左右占比，程序会自动记住。
13. 然后去微信、Word、浏览器或其他软件里粘贴，就会粘贴刚刚恢复的内容。
14. 不想占屏幕时，点悬浮框顶部的「收起」，它会重新变成小圆圈。
15. 展开窗口支持 Windows 原生的最小化、最大化和还原按钮。最大化后再还原时，会自动保持合理的左右区域宽度。
16. 点击右上角 `X` 或按键盘 `Esc`，效果和「收起」相同：窗口会恢复成悬浮球，不会停止监听。
17. 如果连小圆圈也不想显示，点「隐藏」，程序会藏到右下角系统托盘里继续运行。

如果双击后没有出现小圆圈，再双击 `Debug-ClipboardHistory.bat`。这个调试启动会保留黑色窗口，方便查看报错。

## 系统托盘

- 点「隐藏」或右上角关闭按钮，程序不会退出，只会隐藏到右下角系统托盘。
- 左键单击托盘图标，可以直接显示悬浮球。
- 双击托盘图标，可以重新打开历史面板。
- 右键托盘图标，可以选择「打开历史」「显示悬浮球」或「退出」。
- 只有点托盘菜单里的「退出」，程序才会真正停止记录。

## 搜索和预览

- 左上角搜索框可以按文字内容、文件路径、图片尺寸筛选历史记录。
- 左侧卡片列表会直接展示图片缩略图和文字摘要。
- 顶部分类和操作按钮使用小方块图形帮助快速区分；当前分类会以高亮色显示。
- 「图片」栏目会把图片排成矩阵；双指缩放、按住 `Ctrl` 滚动，或使用 `Ctrl` + `+` / `Ctrl` + `-`，可以在“每行 1 张、2 张、3 张……”这些整数档位间切换，并自动填满可用宽度。
- 图片矩阵支持最多每行 12 张图片；小尺寸卡片会自动简化时间文字，避免文字互相挤压。
- 图片矩阵的区域宽度不会跟随缩放跳动；需要改变左右区域占比时，拖动中间分隔条即可。
- 历史列表和图片矩阵使用双缓冲绘制。图片页只加载当前屏幕附近的缩略图，滚动期间会暂停磁盘图片解码，并只刷新对应卡片，以减少卡顿、闪烁和重复加载。
- 右侧区域用于查看当前选中记录的大图或完整文字。
- 触摸板指针放在左侧列表或图片矩阵上做双指缩放，可以调整左侧宽度和卡片/缩略图密度。
- 触摸板指针放在右侧图片上做双指缩放，可以调整图片预览大小。
- 如果你的触摸板没有把缩放手势传给程序，可以按住 `Ctrl`，再用双指上下滑动或鼠标滚轮达到同样效果。

## 多个桌面

Windows 的普通窗口通常只属于一个虚拟桌面。本工具运行时，在任意虚拟桌面快速单独按两次 `Ctrl`，都会把历史窗口移动到当前桌面后再打开。

托盘菜单里的「打开历史」以及双击托盘图标，也会使用相同的当前桌面打开逻辑。

## 位置和大小记忆

- 小圆圈的位置会自动记住。
- 展开的悬浮框位置和大小也会自动记住。
- 这些设置保存在 `data/settings.json`。

## 更换图标

- 悬浮球使用 `app-bubble.png`。这张图应该是透明背景的圆形 PNG。
- 系统托盘使用 `app-icon.png`。
- 如果想换悬浮球，把新的圆形透明 PNG 改名成 `app-bubble.png`，放到这个工具文件夹里覆盖它即可。

## 它保存什么

- 文字：保存文字本身。
- 图片：保存为 `data/images` 里的 PNG 图片。
- 文件：保存你复制的文件路径列表，不会复制文件本体。
- HTML/RTF 富文本：如果系统同时提供普通文字，优先保存普通文字。

## 隐私保护

- 剪贴板历史、复制过的文字、图片、文件路径、本机窗口设置和运行日志都保存在本地 `data` 目录。
- `data/` 已加入 `.gitignore`，不会提交到 GitHub。
- 本机 Codex 配置目录 `.agents/`、`.codex/`，以及日志和临时文件也不会提交。
- GitHub 仓库只包含程序代码、启动脚本、说明文档和应用图标，不包含你的剪贴板历史。

## 自动清理

每次启动时，以及运行期间大约每 10 分钟，它会清理超过 3 天的历史记录。清理范围只限这个工具自己生成的 `data` 目录历史，不会删除你其他文件。

## 注意

小圆圈或悬浮框关掉以后就不会继续记录；下次需要记录时再双击启动即可。
