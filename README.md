# 历史粘贴板

这是一个本地 Windows 小工具：运行后会监听你复制的内容，并把最近 3 天的历史保存在本文件夹的 `data` 目录里。

## 怎么使用

1. 推荐双击 `Start-ClipboardHistory.vbs`，它会静默启动，不显示 PowerShell 黑窗口。也可以双击 `Start-ClipboardHistory.bat` 启动。
2. 启动后屏幕上会出现一个写着「剪」的小圆圈。
3. 平时不用时，让这个小圆圈留在屏幕上即可，它会继续监听你复制的内容。
4. 按住小圆圈可以拖动位置；拖到屏幕边缘附近松手，它会自动吸附成半圆。
5. 单击小圆圈，会展开「历史粘贴板」悬浮框。
6. 你正常复制文字、图片或文件，它会自动出现在左侧黑蓝预览列表；图片会直接显示缩略图，文字会直接显示内容摘要。
7. 想恢复某一条内容时，先点左侧那条记录，再点「复制回剪贴板」。右侧仍然会显示更大的预览。
8. 左侧历史列表和右侧预览区中间有一条竖向分隔条，按住拖动可以调整左右占比，程序会自动记住。
9. 然后去微信、Word、浏览器或其他软件里粘贴，就会粘贴刚刚恢复的内容。
10. 不想占屏幕时，点悬浮框顶部的「收起」，它会重新变成小圆圈。
11. 如果连小圆圈也不想显示，点「隐藏」，程序会藏到右下角系统托盘里继续运行。

如果双击后没有出现小圆圈，再双击 `Debug-ClipboardHistory.bat`。这个调试启动会保留黑色窗口，方便查看报错。

## 系统托盘

- 点「隐藏」或右上角关闭按钮，程序不会退出，只会隐藏到右下角系统托盘。
- 左键单击托盘图标，可以直接显示悬浮球。
- 双击托盘图标，可以重新打开历史面板。
- 右键托盘图标，可以选择「打开历史」「显示悬浮球」或「退出」。
- 只有点托盘菜单里的「退出」，程序才会真正停止记录。

## 搜索和预览

- 左上角搜索框可以按文字内容、文件路径、图片尺寸筛选历史记录。
- 左侧卡片列表会直接展示图片缩略图和文字摘要。
- 右侧区域用于查看当前选中记录的大图或完整文字。
- 触摸板指针放在左侧列表上做双指缩放，可以调整左侧宽度和卡片密度。
- 触摸板指针放在右侧图片上做双指缩放，可以调整图片预览大小。
- 如果你的触摸板没有把缩放手势传给程序，可以按住 `Ctrl`，再用双指上下滑动或鼠标滚轮达到同样效果。

## 多个桌面

Windows 的普通悬浮窗口通常只属于当前虚拟桌面。这个工具隐藏到系统托盘后，托盘图标可以在每个桌面访问，所以这是推荐的跨桌面使用方式。

## 位置和大小记忆

- 小圆圈的位置会自动记住。
- 展开的悬浮框位置和大小也会自动记住。
- 这些设置保存在 `data/settings.json`。

## 更换图标

- 悬浮球使用 `app-bubble.png`。这张图应该是透明背景的圆形 PNG。
- 系统托盘使用 `app-icon.png`。
- 如果想换悬浮球，把新的圆形透明 PNG 改名成 `app-bubble.png`，放到这个工具文件夹里覆盖它即可。

## 它保存什么

- 文字：保存文字本身。
- 图片：保存为 `data/images` 里的 PNG 图片。
- 文件：保存你复制的文件路径列表，不会复制文件本体。
- HTML/RTF 富文本：如果系统同时提供普通文字，优先保存普通文字。

## 自动清理

每次启动时，以及运行期间大约每 10 分钟，它会清理超过 3 天的历史记录。清理范围只限这个工具自己生成的 `data` 目录历史，不会删除你其他文件。

## 注意

小圆圈或悬浮框关掉以后就不会继续记录；下次需要记录时再双击启动即可。
